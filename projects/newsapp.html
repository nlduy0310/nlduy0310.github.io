PROJECT: News Site

DESCRIPTION

Final project of Web Development class - a news website.

This covers all the basics plus: authorization, full text search and intergration of some common technologies like reCAPTCHA, TinyMCE, Cloudinary, Mailjet

APP TYPE

Website / Academic

TECHNOLOGIES USED

  • Handlebars + HTML/CSS/JS + Bootstrap for UI.

  • ExpressJS for backend.

  • PostgreSQL/Sequelize & Redis for data.

  • Integration of reCAPTCHA, Mailjet, Cloudinary, TinyMCE.

MEMBERS

Me Hoi Phu

FEATURES

  • User management: sign in, sign up, reset password, update personal information, extend "premium" status.

  • Display trending articles and each article in details. View articles list by category or tag. Search on articles' content and sort them by different criteria.

  • Functionalities for different user types:

    • Normal user: be able to comment on articles

    • Premium user: be able to view premium articles

    • Writer: Be able to create a new article via a WYSIWYG editor, edit existing/rejected articles and submit them for review

    • Editor: In charge of a specific category. Review an article and decide to reject/accept it. Decide the final categories/tags/publish date of an article.

    • Admin: Manage categories/tags/articles/users list. Extend a user's premium status and assign categories for editors.

MY CONTRIBUTION

  • Wrote the UI template for sign in & sign up pages, writer page, including reCAPTCHA intergration.

  • Crawled sample data & prepared database seeders.

  • Handled search post by title/content or full text search, extend premium feature.

  • Handled functionalities regarding the writer role, including: new/edit article page, "my articles" page, WYSIWYG editor's content and image upload using Cloudinary.
