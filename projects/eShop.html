PROJECT: eShop

DESCRIPTION

My first ever web project - an online clothing store

This covers the basics of web development: authentication, sessions, requests validation, routing, CRUD, server side templating.

APP TYPE

Website / Academic

TECHNOLOGIES USED

  • Handlebars + HTML templates for UI

  • ExpressJS for backend

  • PostgreSQL/Sequelize & Redis for data

  • Mailjet intergration

MEMBERS

Me

FEATURES

  • Register, login, logout & reset password.
  • View products, sort by category/brand/tag(s).
  • Leave a review and rate a product. Add an item to your wish list.
  • User can add multiple items to cart, add an address and place the order.
