PROJECT: Movies Review Site

DESCRIPTION

Final project of Software Design class - a movie review site similar to IMDB.

APP TYPE

Website / Academic

TECHNOLOGIES USED

  • Handlebars & HTML templates for UI.

  • ExpressJS for backend.

  • PostgreSQL/Sequelize & Redis for data.

  • Python/PyTorch/FastAPI for training & deploying a sentiment analysis model.

MEMBERS

Me Phu Qui

FEATURES

  • Suggest movies based on ratings or personal interest.

  • Show details of a movie, rate, comment or share to Facebook.

  • Analyse movie reviews using a PyTorch Sentiment Analysis model.

  • Search & sort movies by category/text/voice.

  • User management: sign in, sign up, update information, Facebook login.

MY CONTRIBUTION

  • Crawled and prepared database seeders.

  • Handled search options.

  • Facebook login.

  • Handled movie details page.
